●プログラム実行について

秀丸では標準入力、標準出力を利用するプログラムを利用して、テキストを加工することができます。

昔はlinuxのような便利なコマンドが少なくて活用しづらい機能でしたが、現在はWindowsでも便利なコマンドが増えたりPowerShellが導入されたことで色々な編集に活用することができます。

【１】 プログラム実行の方法
【２】 実用例

【１】プログラム実行の方法

メニューから「その他→プログラム実行」できますが、そのときコマンドラインテキストの最後に「 <con >con」と付けて実行することで、外部プログラムの標準入力に現在の選択範囲テキストを渡し、標準出力を選択範囲に出力することができます。

たとえばある行範囲を選択して「その他→プログラム実行」から「sort <con >con」を実行することで、選択範囲を並び替えた状態にテキストを加工することができます。

【２】実用例

sort <con >con

選択範囲を並び替える。

findstr "hoge" <con >con

hogeという文字が入った行だけを抜き出す。

powershell -c "$input | Get-Unique" <con >con

選択範囲の重複行を削除する。

powershell -c "1..5" >con

「1 <改行> 2 <改行> 3 <改行> 4 <改行> 5」という連番を選択範囲に書き出す。

cmd /c "dir /b c:\hoge" >con

「c:\hoge」フォルダのファイル一覧を選択範囲に書き出す。
