![Hidemaru Q and A](hide_by2.png)\


##はじめに
秀丸エディタＱ＆Ａ集の8版です。
対象は以下になります。

*   秀丸エディタVer.8以降
*   OSはWindows Vista以降

現状では、第7版の内容を、対象の環境に合わせ更新した程度です。
Ver.8までに追加された機能に関してはこれから。
