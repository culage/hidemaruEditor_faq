●サブルーチンからの戻り値(数値)

戻り値を数値で得たいのですが。

サブルーチンからの戻り値は文字列型変数だけでなく、数値型でも得られます。 次の例は上の例とほぼ同じですが、戻り値を数値型で返してします。

//------- サンプルマクロ -------//
$a="123";
$b="456";
call calcStrToNum $a,$b;
#ansNum=##return;
endmacro;

calcStrToNum:
    $$s1=$$1;
    $$s2=$$2;
    ##wa=val($$s1)+val($$s2);
    return ##wa;
// ---------------------------- //
